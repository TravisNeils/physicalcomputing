For this assignment I wanted to test out how to use the Multicolored LED to do this I connected each lead to the corresponding color of LED so that when I suplied power to that lead we could see what the inputs are. This lets us see exactly what different color combinations make. For example red and green make yellow.

The red and green LEDs have a low enough voltage drop that two can be run in series and have enough current to be bright The blue LEDs have a much higher voltage drop around 3 V so I need to run them in parallel. I used 330 ohm resistors for all three LEDs to reduce the current and brighness a little bit more so that it was easier to look at. To ensure that my LEDs wouldn't burn out I use a voltage drop of 1.7 for red and yellow and 2.5 for blue and ohms law. For red and yellow 5 - 1.7 = 0.03 amps * R_min R_min = 110, so 330 would be safe to use. For blue 5 - 2.5 = 0.03 amps * R_min, R_min = 83 so my resistance of 330 + 100 = 430 ohms would be more than enough. Having this reduced current would make the LEDs dimmer and more viewable.

In arduino code I ran through all the possible color combinations with nested for loops. It was hard to comment these well. If we start from the very inner loop of the red LED the for loop will turn it on and off. The loop outside of that will run that loop twice, once with the green LED on and once with the green LED off. The final loop ouside of that will run that whole thing twice aswell once with the blue on and once with it off.

            const int ledPin =  8;

            const int buttonPin = 3;
            boolean aborted = false;
            boolean button_pressed = false;

            void setup() {
              // put your setup code here, to run once:
              // Sets up pins 5 through 12 as output pins.
              for (int i = 5; i < 11; i++){
                pinMode(i, OUTPUT);
              }
              // initialize the pin wired to the button as an input:
              pinMode(buttonPin, INPUT);
            
            }
            
            void loop() {
              // put your main code here, to run repeatedly:
              delay(random(2000)); // Make the time between sequences random between 0 and 2 seconds. 
              countdown(); // Runs the countdown function that triggers yellow LEDs.
               // When the player has to press the button
              button_pressed = false; // every loop will reset if the button was pressed in the last loop.
              // This foor loop will read the input from the button every 10 miliseconds for 0.8 seconds
              // this is when the button must be pressed to win the game if button is pressed trigger the things:
              for (int i = 8; i < 80; i = i + 1){
                if (digitalRead(buttonPin) == HIGH){
                  button_pressed = true;  
                }
                delay(10);
              }
              //Checks if both the button was pressed during the required time and not before.
              // Aborted would be true if the player pressed the button while a yellow LED is still on.
              if (button_pressed & !aborted){
                fade(green); // Tells the player they won by lighting and fading green led
              } else {
                fade(red); // Tells the player they lost by lighting and fading green led
              }
              aborted = false;
            } 
            
            void countdown() {
              // This loop runs the column of yellow LEDs counting down to the button
              for (int pin = 8; pin < 12; pin = pin + 1){
                digitalWrite(pin - 1, LOW); // turns the last LED off to provide moving light effect
                delay(10); // Delay to prevent damage
                digitalWrite(pin, HIGH); // Turns the LED on in sequence to provide effect
                delay(100); // Gives the player some time to see the LED
                // If the button is pressed set let the game know not to let them win and cancel the LED sequence.
                if(digitalRead(buttonPin) == HIGH){
                  aborted = true;
                  break;
                }
              }
            }
            
            void fade(int pin){
              // Start with full bightness of the pin then quickly decrease it 
              for (int brightness = 255; brightness > 20; brightness = brightness - 1){
                analogWrite(pin, brightness); // set the led brighness level
                delay(5);
              }
              digitalWrite(pin, LOW); // Turn the LED off when done. 
            }
